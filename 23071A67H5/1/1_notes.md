## Notes

- The webpage layout is created using the `<frameset>` tag.  
- The browser window is divided into three horizontal frames.  
- The top frame is used as the navigation frame.  
- The navigation frame contains links to different book categories.  
- The middle frame is used as the content display frame.  
- Book details are displayed in the content frame.  
- The `target` attribute is used to load pages into the content frame.  
- The footer frame is placed at the bottom of the page.  
- The footer frame displays additional information.  
- Only the content frame updates when a navigation link is clicked.
